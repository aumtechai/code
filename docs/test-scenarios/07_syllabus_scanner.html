🔳 GAP-002 — Syllabus → Calendar Write Integration

The Syllabus Scanner successfully parses a PDF and extracts dates, but the extracted events are not persisted to the calendar. The "Add to Calendar" button renders in the UI but does not call the backend.

Root cause: POST /api/calendar/events exists for manual event creation but SyllabusScanner.jsx does not call it after parse. The handleAddToCalendar() function is a no-op placeholder.

To implement: After parse, map each event { date, title, type } → POST /api/calendar/events with { title, date, category: "syllabus", course_id }. Then refresh the schedule component. Estimated effort: ~2 hours.
