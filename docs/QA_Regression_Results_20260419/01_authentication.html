AUTH-BUG-001: 401 interceptor clears login error on login page
In api.js, the response interceptor catches all 401s and does window.location.href = '/login', which reloads the page and clears the React error state. Fix: guard with if (!window.location.pathname.includes('/login')).
api.js · Line 57-60

AUTH-BUG-002: Trailing whitespace in GOOGLE_CLIENT_ID env var
.env.local has GOOGLE_CLIENT_ID="413983...62u7.apps.googleusercontent.com \r\n" — the trailing space can cause Google token verification to fail locally.
